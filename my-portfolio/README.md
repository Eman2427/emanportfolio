# Eman's Portfolio

## Run in VS Code

1. Unzip and open the `my-portfolio` folder in VS Code
2. Open terminal: Ctrl + `
3. Run:

   npm install
   npm run dev

Browser opens at http://localhost:5173

## Deploy to Vercel

1. Push to GitHub
2. Import on vercel.com
3. Click Deploy (Vercel auto-detects Vite)

Requires Node.js from https://nodejs.org (LTS version)
